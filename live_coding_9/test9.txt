Задача: "найти место для вставки"

В функцию searchInsert  передается отсортированный массив целых
положительных чисел nums. А так же мы получаем искомое
целое положительное число target.

Найти индекс, на котором находится наше число. Если в нашем
массиве такого числа нет, то вернуть индекс, где бы это
число могло бы быть

Важное замечание: алгоритм должен работать со скороситью O(log n)

Оригинал задачи:
https://leetcode.com/problems/search-insert-position

Пример 1:
               0 1 2
Input: nums = [1,3,5,6], target = 5
Output: 2

Пример 2:
               0  1  2 3 4
Input: nums = [1 <2> 3,5,6], target = 2
Output: 1

Пример 3:
               0 1 2 3  4
Input: nums = [1,3,5,6,<7>], target = 7
Output: 4

1 <= длина числа <= 104
-104 <= nums[i] <= 104
nums содержит различные значения, отсортированные в порядке возрастания.
-104 <= target <= 104
